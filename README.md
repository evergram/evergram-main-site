# evergram-main-site
Evergram main website


##Install Node.JS

```
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.24.0/install.sh | bash

nvm install stable
```

###Clone

```
git clone git@github.com:evergram/evergram-main-site.git
```


###Init

```
cd evergram-main-site
npm install
```
